Tertarik Belajar Web Programming, Silahkan daftar

HTML

HTML adalah tag tag yang digunakan untuk membuat kerangka dari sebuah Web seperti Text, audo, video maupun foto

CSS

CSS adalah script-script yang digunakan untuk memperindah tampilan dari Web seperti pengaturan warna, teks, dan layout sebuah web

Javascript

Javascript adalah bahasa pemrograman yang dapat membuat webiste menjadi interaktif

PHP

PHP adalah bahasa pemrograman untuk mengatur interaksi Web dengan User

Laravel

Laravel adalah framework MVC untuk PHP. Memudahkan developer dalam memisahkan font-end dan back-end
